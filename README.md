# G-Navigator AI

AI가 항로부터 리스크까지 판단하는 글로벌 물류 의사결정 플랫폼

*Navigate Every Vehicle, Anywhere.*

## 이게 왜 다른가

기존 TMS/WMS는 데이터를 "조회"만 해줍니다. AIS 트래킹 서비스는 위치만, 화주-포워더 매칭 서비스는 견적 비교만 해줍니다. 어떤 경로와 전략을 선택해야 하는가에 대한 답은 여전히 담당자의 감과 경험에 의존합니다.

G-Navigator AI는 경로·비용·리스크·통관 정보를 하나의 화면에 통합하고, 몬테카를로 시뮬레이션과 생성형 AI(Gemini)로 실행 가능한 판단까지 제시합니다.

**기존 서비스 대비 차이점**

- 정보 제공: 조회만 가능 → 조회 + 판단까지 제공
- 리스크 대응: 사후 확인 → 사전 예측 + 자동 대안 제안
- 데이터 통합: 시스템별로 분산 → 경로·비용·리스크·통관을 한 화면에서
- 접근성: 고가/대형 화주 중심 → 웹 기반, 누구나 즉시 사용 가능

## 핵심 기능

**01. Global Route Command Center**
다중 경유지 경로 설계, AI 최적 항로 자동 생성, 실시간 AIS 선박 추적(국내 60척·국제 150척 규모), 항만 혼잡도·기상(태풍) 리스크 지도 시각화

**02. AI Strategy & Decision Hub**
전략 A/B/C(신속·균형·경제) 자동 생성 및 동시 비교, 운송 수락·조건부·비추천 판단 AI, 자연어 질의응답 및 AI 리스크 브리핑(Gemini)

**03. Risk & Cost Intelligence**
몬테카를로 시뮬레이션(1,000회) 기반 비용·납기 예측, 통관일수·HS 관세·CBAM·보험료 자동 계산, 리스크 임계치 초과 시 대안 경로 자동 제안

**04. Insights & Collaboration**
진행중 배송 현황판 및 운행 패턴 분석, PDF·Excel 리포트 생성, 이메일·Slack 공유, 즐겨찾기 경로 저장 및 경로 비교

## AI를 쓰는 방식

숫자로 정확히 계산 가능한 문제는 시뮬레이션·통계가, 언어를 이해하고 요약하고 자유롭게 답해야 하는 문제만 생성형 AI가 처리합니다. AI를 모든 곳에 붙이는 대신, AI가 실제로 가치를 더하는 지점에만 정확히 배치했습니다.

- 생성형 AI(Gemini) 담당: 자연어 질의응답, AI 리스크 브리핑(뉴스 요약), 운행 패턴 인사이트
- 규칙·통계 담당: 전략 A/B/C 추천(몬테카를로 시뮬레이션), 운송 수락 판단(확률 계산), 리스크 자동 경고(임계값 조건문)

## 기대 효과

- **의사결정 방식 전환**: 뉴스·항만혼잡도·환율·통관·리스크를 개별 조회하던 방식에서 하나의 대시보드로 통합
- **리스크 대응**: 사후 대응에서 사전 예측으로, 임계치 초과 시 대안 경로 자동 제안
- **비용 절감 가능성**: 전략별 비교로 상황에 맞는 최적 운송 조합 선택 지원
- **환경**: 도로·철도 탄소배출량 비교 제공, 저탄소 운송수단 전환 시 절감 효과 사전 확인

## Tech Stack

Python · Streamlit · Folium · Altair · Google Gemini API · SQLite

**외부 연동** (API 키 등록 시 실데이터로 자동 전환, 없으면 시뮬레이션 데이터로 폴백)

- AISHub: 실시간 선박 AIS 데이터
- 기상청(공공데이터포털): 태풍 경로
- Frankfurter: 실시간 환율
- 관세청 unipass: HS코드 관세율
- Google Gemini: AI 어시스턴트 및 브리핑

## Quick Start

```
git clone https://github.com/suhhyunu/g-navigator-ai.git
cd g-navigator-ai
pip install -r requirements.txt
streamlit run atlas_ai_merged.py
```

API 키 없이도 모든 기능이 시뮬레이션 데이터로 즉시 동작합니다.

## 사용법

1. 사이드바에서 언어(한/영)와 API 키(선택)를 설정합니다.
2. 국내운송 / 국제운송 탭에서 출발지·도착지·화물 정보를 입력합니다.
3. AI 추천 전략(A/B/C) 중 하나를 선택하고 시뮬레이션을 실행합니다.
4. 결과를 PDF/Excel로 내보내거나 이메일/Slack으로 공유합니다.
5. 홈 대시보드 탭에서 진행중인 배송 현황과 운행 패턴을 확인합니다.

## 참고 사항

- API 키가 없어도 모든 기능이 시뮬레이션 데이터로 동작합니다.
- SQLite DB(atlas_ai_data.db)는 로컬에 자동 생성되며, 즐겨찾기·시뮬레이션 이력·배송 현황이 저장됩니다.
- 항만 혼잡도·빈 화차 정보 등 일부 데이터는 실시간 공개 API가 없어 추정치로 대체되어 있습니다 (코드 주석 참고).
